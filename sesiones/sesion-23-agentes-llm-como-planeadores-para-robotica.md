# Sesión 23: Agentes LLM como planeadores para robótica

**Semana:** 5  
**Fecha:** 24 de Junio  
**Tema GEM:** Preparación para el futuro  
**Subtema GEM:** Toma de decisiones

## Objetivo de la sesión

Desarrollar los elementos conceptuales y prácticos relacionados con **Agentes LLM como planeadores para robótica**, conectándolos con el proyecto integrador de IA con LLM para automatización y robótica.

## Contenido del profesor

Explicación del ciclo percepción-decisión-acción, metas en lenguaje natural, descomposición de tareas, replanteamiento con feedback y auditoría de decisiones.

Explicación del ciclo cerrado con agente: meta natural → plan → acción → feedback → replanteamiento; criterios de éxito, falla y parada segura.

## Actividad del alumnado

Diseñar el planificador del agente para una tarea robótica o automatizada del proyecto.
