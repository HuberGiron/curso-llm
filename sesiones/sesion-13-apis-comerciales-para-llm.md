# Sesión 13: APIs comerciales para LLM

**Semana:** 3  
**Fecha:** 10 de Junio  
**Tema GEM:** Preparación para el futuro  
**Subtema GEM:** Reducción de la incertidumbre / Evaluación de opciones estratégicas

## Objetivo de la sesión

Desarrollar los elementos conceptuales y prácticos relacionados con **APIs comerciales para LLM**, conectándolos con el proyecto integrador de IA con LLM para automatización y robótica.

## Contenido del profesor

Comparación de Gemini, Grok, Claude y alternativas comerciales frente a modelos locales; API keys, secretos, cuotas, costos, disponibilidad, calidad y riesgos.

Explicación de backend intercambiable: variables de entorno, cliente Python, mock seguro y estrategia para alternar entre Ollama y APIs comerciales.

## Actividad del alumnado

Elaborar una matriz comparativa de modelo local vs API comercial aplicada al proyecto.
