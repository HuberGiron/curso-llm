# Sesión 04: Benchmark con Python y Ollama

**Semana:** 1  
**Fecha:** 28 de Mayo  
**Tema GEM:** Aplicación tecnológica  
**Subtema GEM:** Conceptos / Herramientas / Casos

## Objetivo de la sesión

Desarrollar los elementos conceptuales y prácticos relacionados con **Benchmark con Python y Ollama**, conectándolos con el proyecto integrador de IA con LLM para automatización y robótica.

## Contenido del profesor

Sesión principalmente práctica. El profesor acompaña la implementación, resuelve dudas técnicas y verifica evidencias mínimas.

## Actividad del alumnado

Ejecutar un script de benchmark y generar una tabla comparativa para justificar la selección de modelo.
