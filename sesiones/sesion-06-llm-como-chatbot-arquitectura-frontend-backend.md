# Sesión 06: LLM como chatbot: arquitectura frontend/backend

**Semana:** 2  
**Fecha:** 1 de Junio  
**Tema GEM:** Aplicación tecnológica  
**Subtema GEM:** Conceptos / Herramientas / Casos

## Objetivo de la sesión

Desarrollar los elementos conceptuales y prácticos relacionados con **LLM como chatbot: arquitectura frontend/backend**, conectándolos con el proyecto integrador de IA con LLM para automatización y robótica.

## Contenido del profesor

Explicación de arquitectura cliente-servidor para chatbot local: HTML/CSS/JS, Python, API de Ollama, parámetros de generación y registro de métricas.

Demostración guiada de backend Python y frontend web para conversar con un LLM local; estructura de endpoint, solicitud y respuesta del modelo.

## Actividad del alumnado

Dibujar el diagrama del flujo de datos del chatbot y ubicar frontend, backend, modelo local, parámetros y métricas.
