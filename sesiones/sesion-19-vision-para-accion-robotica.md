# Sesión 19: Visión para acción robótica

**Semana:** 4  
**Fecha:** 18 de Junio  
**Tema GEM:** Aplicación tecnológica  
**Subtema GEM:** Conceptos / Herramientas / Casos

## Objetivo de la sesión

Desarrollar los elementos conceptuales y prácticos relacionados con **Visión para acción robótica**, conectándolos con el proyecto integrador de IA con LLM para automatización y robótica.

## Contenido del profesor

Sesión principalmente práctica. El profesor acompaña la implementación, resuelve dudas técnicas y verifica evidencias mínimas.

## Actividad del alumnado

Convertir una imagen o estado visual en salida JSON y acción/recomendación del sistema.

## Agenda sugerida de 3 horas

| Tiempo | Actividad |
|---|---|
| 0:00–0:20 | Repaso del objetivo y criterios de evidencia |
| 0:20–0:50 | Demostración técnica del profesor |
| 0:50–1:40 | Implementación individual o por equipo |
| 1:40–2:30 | Pruebas, medición y depuración |
| 2:30–3:00 | Documentación inicial y cierre |

## Evidencia mínima
