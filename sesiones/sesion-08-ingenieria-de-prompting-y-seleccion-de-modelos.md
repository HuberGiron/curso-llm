# Sesión 08: Ingeniería de prompting y selección de modelos

**Semana:** 2  
**Fecha:** 3 de Junio  
**Tema GEM:** Preparación para el futuro  
**Subtema GEM:** Reducción de la incertidumbre / Evaluación de opciones estratégicas

## Objetivo de la sesión

Desarrollar los elementos conceptuales y prácticos relacionados con **Ingeniería de prompting y selección de modelos**, conectándolos con el proyecto integrador de IA con LLM para automatización y robótica.

## Contenido del profesor

Explicación de buenas prácticas de prompting: rol, contexto, tarea, formato, restricciones, ejemplos, salidas JSON y criterios para elegir modelo.

Explicación de validación de prompts con instrucciones ambiguas, JSON estructurado, temperatura, límites de tokens y fallas comunes.

## Actividad del alumnado

Rediseñar prompts para convertir instrucciones humanas en comandos robóticos estructurados.
