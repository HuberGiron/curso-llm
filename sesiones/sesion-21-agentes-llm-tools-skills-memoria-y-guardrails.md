# Sesión 21: Agentes LLM: tools, skills, memoria y guardrails

**Semana:** 5  
**Fecha:** 22 de Junio  
**Tema GEM:** Métodos relacionados con los estudios sobre el futuro  
**Subtema GEM:** Planeación estratégica / Previsión estratégica

## Objetivo de la sesión

Desarrollar los elementos conceptuales y prácticos relacionados con **Agentes LLM: tools, skills, memoria y guardrails**, conectándolos con el proyecto integrador de IA con LLM para automatización y robótica.

## Contenido del profesor

Explicación de diferencias entre chatbot, herramienta, skill y agente; planner/executor, memoria, validadores, logs, límites de actuación y seguridad.

Demostración de agente mínimo con funciones para mover, consultar estado y registrar eventos.

## Actividad del alumnado

Identificar las skills necesarias para el proyecto y proponer guardrails, validadores y registros de decisión.
