# Rúbrica del proyecto

| Criterio | Excelente | Suficiente | Insuficiente |
|---|---|---|---|
| Prospectiva | Escenario claro, fuentes formales, riesgos y oportunidades bien argumentadas | Escenario general con pocas fuentes | No hay análisis prospectivo |
| Arquitectura | Diagrama completo, flujo claro, validadores y feedback | Arquitectura funcional pero incompleta | Flujo confuso o no reproducible |
| LLM/agente | Prompt/agente robusto, salida estructurada y manejo de errores | Funciona en casos básicos | Salida no validable o inestable |
| Integración | Comunicación funcional con robot/simulador/embebido | Integración parcial | No hay integración demostrable |
| Medición | Métricas, tablas, logs y análisis | Métricas limitadas | Sin medición |
| Documentación | Repositorio reproducible con evidencias | Documentación parcial | No permite reproducir |
| Demo | Demo clara, estable y argumentada | Demo básica | Demo no funcional |

## Checklist de entrega final

- [ ] Repositorio público o compartido.
- [ ] Reporte académico en PDF o Markdown.
- [ ] Video corto de demostración.
- [ ] Presentación final.
- [ ] Código fuente.
- [ ] Instrucciones de instalación y ejecución.
- [ ] Evidencia de pruebas.
- [ ] Matriz de riesgos, costos y recursos.
- [ ] Reflexión prospectiva.
