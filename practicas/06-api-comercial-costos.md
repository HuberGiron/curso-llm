# Práctica 6. API comercial y estimación de costos

## Objetivo

Usar una API comercial de forma segura y estimar costos de operación.

## Contexto técnico

La práctica enfatiza seguridad de secretos, costos por tokens y comparación local vs nube.

## Material necesario

- Computadora con Python 3.10 o superior.
- Editor de código.
- Git y repositorio personal.
- Ollama o API indicada según la práctica.
- Dependencias listadas en el código correspondiente.

## Procedimiento sugerido

1. Crea `.env`.
2. Ejecuta una llamada de prueba.
3. Calcula tokens aproximados.
4. Estima costos para tres escalas.
5. Compara con Ollama.
