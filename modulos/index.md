# Módulos académicos

Estos módulos sustituyen a una presentación tradicional. Cada página está escrita como material de lectura y como guion técnico para clase: contexto académico, conceptos, arquitectura, procedimiento, ejemplos, código, ejercicios y lecturas.

No se recomienda leerlos como cronograma. La lógica es de manual técnico-académico: primero se entiende el fenómeno tecnológico, después se implementa, se mide y finalmente se discuten sus implicaciones prospectivas.
